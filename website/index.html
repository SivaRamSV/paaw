Why PAAW?

🧠

Mental Model

PAAW builds a knowledge graph about you — your work, preferences, relationships, and goals. Context that persists across conversations.

🔒

Privacy First

Self-hosted. Your data never leaves your machine. Use any LLM provider — Claude, GPT, or local models via Ollama.

⏰

Scheduled Jobs

Set up automated tasks that run on schedule — morning news briefings, investment monitoring, birthday reminders, and more.

🔧

Tool Integration

Connect to web search, Discord, and any MCP-compatible tool. PAAW can search the web, send messages, and take actions.

💬

Multi-Channel

Chat via Web UI, CLI, or Discord. Talk to PAAW from your phone while commuting. Same context everywhere.

🎯

Goal Oriented

PAAW doesn't just answer questions — it helps you track goals, remember commitments, and stay on top of what matters.

Get Started in 3 Steps

1

Clone & Configure

Clone the repository and add your LLM API key.

git clone https://github.com/SivaRamSV/paaw.git
cd paaw
2

One-Command Start

The startup script handles everything — Docker, database, search engine.

./start.sh
3

Start Chatting

Open the web UI and introduce yourself. PAAW will remember.

http://localhost:8080
